# The Reflection Machine 🎰

A vintage slot machine for daily reflection. Pull the lever to get two journaling prompts; ring the bell to draw a weekly research topic.

## Features
- 120+ thought-provoking prompts about life, emotion, curiosity, and growth
- Vintage slot-machine UI with lever animation and reel spin
- Built-in journal that auto-saves to your browser (localStorage)
- Download today's entry as `.txt`
- Weekly research word — a topic to chase down rabbit holes
- Pure HTML / CSS / JS — no build step, deploys instantly to Vercel

## Run locally
Just open `index.html` in your browser, or:
```bash
python3 -m http.server 5173
# then visit http://localhost:5173
```

## Deploy
```bash
vercel
```

## Privacy
Everything lives in your browser. No backend, no tracking, no uploads.
